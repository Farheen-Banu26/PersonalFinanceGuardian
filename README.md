Project Overview 

FinanceTracker is a user-friendly Personal Finance Management System designed to help individuals track expenses, set savings goals, and make smarter financial decisions.
Built with a clean, modern UI and gamified elements, the system encourages financial discipline through goal tracking, streaks, and insightful visual analytics.

FinanceTracker is a fully frontend-driven personal finance management web application built using TypeScript, React, Vite, and Tailwind CSS.
It provides users with an intuitive platform to:

Track daily expenses

Create and manage savings goals

Monitor financial habits

Visualize insights through clean charts and dashboards

The application focuses on simplicity, speed, and clarity, offering users a smooth experience while understanding and managing their finances.
